# Silver Care MCP - 온열질환 예방 서비스

고령자를 위한 실시간 온열질환 위험도 분석 및 무더위 쉼터 안내 MCP (Model Context Protocol) 서버입니다.

---

## 기능

- 🌡️ **실시간 날씨 분석**: 위치 기반 온열질환 위험도 평가
- 🏠 **무더위 쉼터 찾기**: 주변 경로당, 주민센터 등 쉼터 안내
- 🆘 **긴급 구조 요청**: SOS 메시지 생성 및 119 신고 링크 제공
- 📍 **위치 변환**: 위치 이름을 위도/경도로 자동 변환

---

## 제공하는 Tools

1. **`geocode_location`**: 위치 이름(예: '서울시 종로구')을 위도/경도로 변환
2. **`analyze_heat_risk`**: 실시간 날씨 기반 온열질환 위험도 분석
3. **`find_cooling_shelter`**: 주변 무더위 쉼터 찾기
4. **`generate_sos_alert`**: 긴급 구조 요청 메시지 생성

---

## Tool 사용 예시

### 예시 1: 날씨 분석

**사용자 입력**: "서울시 종로구 날씨가 어때? 위험한가요?"

**Tool 호출**:
```json
{
  "name": "analyze_heat_risk",
  "arguments": {
    "location": "서울시 종로구"
  }
}
```

**응답**: 현재 기온, 체감온도, 위험 단계, 행동 가이드 제공

### 예시 2: 쉼터 찾기

**사용자 입력**: "부산 해운대 근처에 무더위 쉼터 있나요?"

**Tool 호출**:
```json
{
  "name": "find_cooling_shelter",
  "arguments": {
    "location": "부산 해운대"
  }
}
```

**응답**: 주변 무더위 쉼터 목록과 카카오맵 길찾기 링크 제공

### 예시 3: 긴급 상황

**사용자 입력**: "지금 어지러워요. 제주시청 근처인데 도와주세요"

**Tool 호출**:
```json
{
  "name": "generate_sos_alert",
  "arguments": {
    "location": "제주시청",
    "symptoms": "어지러움"
  }
}
```

**응답**: 구조 요청 메시지와 119 신고 링크 제공

### 위치 입력 방식

각 Tool은 다음 두 가지 방식으로 위치를 받을 수 있습니다:

1. **위치 이름 (텍스트)**: `{"location": "서울시 종로구"}`
2. **위도/경도 (숫자)**: `{"lat": 37.5665, "lon": 126.9780}`

위치 이름을 입력하면 자동으로 위도/경도로 변환됩니다.

## 사용자 가이드

### 서비스 소개

Silver Care MCP는 고령자를 위한 실시간 온열질환 위험도 분석 및 무더위 쉼터 안내 서비스입니다. 사용자의 위치를 기반으로 현재 날씨 정보를 분석하고, 온열질환 위험도를 평가하며, 주변 무더위 쉼터를 찾아드립니다.

### 사용 방법

#### 위치 입력 방식

서비스를 사용할 때는 다음 두 가지 방식으로 위치를 입력할 수 있습니다:

1. **위치 이름 (텍스트)**: 
   - 예: "서울시 종로구", "부산 해운대", "제주시청"
   - 가장 간단하고 편리한 방법

2. **위도/경도 (숫자)**: 
   - 예: 위도 37.5665, 경도 126.9780
   - 정확한 좌표가 필요한 경우 사용

#### 자연어로 질문하기

AI와 대화하듯이 자연스럽게 질문하면 됩니다:

- "서울시 종로구 날씨가 어때? 위험한가요?"
- "부산 해운대 근처에 무더위 쉼터 있나요?"
- "지금 어지러워요. 제주시청 근처인데 도와주세요"

### 위험도 단계 설명

서비스는 체감온도를 기반으로 다음 4단계로 위험도를 평가합니다:

#### 🚨 위험 (Extreme Danger)
- **체감온도**: 38°C 이상
- **설명**: 체온이 급격히 오를 수 있습니다. 즉시 야외 활동을 중단하세요.
- **행동**: 119에 신고하거나 시원한 곳으로 즉시 이동하세요.

#### ⚠️ 경고 (Danger)
- **체감온도**: 35°C 이상
- **설명**: 현기증이나 두통이 발생할 수 있습니다.
- **행동**: 가장 가까운 무더위 쉼터로 이동하고, 옷을 헐렁하게 하세요.

#### ⚡ 주의 (Caution)
- **체감온도**: 31°C 이상
- **설명**: 장시간 햇볕 노출 시 위험합니다.
- **행동**: 30분마다 그늘에서 쉬고 물을 자주 드세요.

#### ✅ 관심 (Attention)
- **체감온도**: 31°C 미만
- **설명**: 야외 활동하기 비교적 괜찮습니다.
- **행동**: 목이 마르지 않아도 물을 규칙적으로 드세요.

### 자주 묻는 질문

#### Q1: 어떤 위치를 입력할 수 있나요?

**A**: 한국의 모든 지역을 입력할 수 있습니다. 예:
- 시/도 단위: "서울시", "부산시", "제주도"
- 구/군 단위: "서울시 종로구", "부산시 해운대구"
- 상세 주소: "서울시 종로구 종로3가"
- 유명 장소: "제주시청", "해운대 해수욕장"

#### Q2: 정확한 위치를 모르면 어떻게 하나요?

**A**: 대략적인 위치만 입력해도 됩니다. 예를 들어 "서울시 종로구"만 입력해도 해당 지역의 날씨 정보를 제공합니다.

#### Q3: 무더위 쉼터는 어떻게 찾나요?

**A**: 위치를 입력하면 자동으로 주변의 무더위 쉼터를 찾아드립니다. 각 쉼터의 거리, 종류, 카카오맵 길찾기 링크를 제공합니다.

#### Q4: 긴급 상황 시 어떻게 사용하나요?

**A**: 현재 위치와 증상을 말씀해주시면 구조 요청 메시지가 생성됩니다. 이 메시지를 보호자나 119에 전달하시면 됩니다.

#### Q5: 서비스는 무료인가요?

**A**: 네, 무료로 사용할 수 있습니다.

#### Q6: 개인정보가 저장되나요?

**A**: 아니요. 위치 정보는 요청 시에만 사용되며 저장되지 않습니다.

### 주의사항

1. **긴급 상황**: 이 서비스는 정보 제공용입니다. 생명이 위급한 상황에서는 즉시 119에 신고하세요.

2. **위치 정확도**: 대략적인 위치만 입력해도 사용 가능하지만, 더 정확한 정보를 원하시면 상세 주소를 입력하세요.

3. **날씨 정보**: 실시간 날씨 정보는 외부 API를 통해 제공되며, 약간의 지연이 있을 수 있습니다.

4. **쉼터 정보**: 현재는 주요 지역의 쉼터 정보를 제공하며, 모든 지역의 쉼터 정보가 포함되지 않을 수 있습니다.
